# O MOLHO SECRETO — por que o estudo de 3.724 acertou a mão (e nada depois acertou)

> Engenharia reversa dos artefatos de janeiro (522 v1 → 3.724 final → funil →
> PDF rejeitado → deck) + a reação dela ("gostei SÓ desse claude") + as críticas
> de 22/07 ("mapeamento, não roteiros"; "sem pé nem cabeça"). Isto é a RECEITA —
> vira o default do commentarium (v0.7).

## Os 8 elementos (nenhum é opcional)

1. **ELA É A PESQUISADORA, não a opinadora.** O documento a transforma em dona
   de um dado proprietário ("EU analisei 3.724 comentários"). Metodologia +
   limitações declaradas = credencial. É por isso que ela chamou de "minha arte":
   o estudo é um ATIVO dela, não um conselho nosso.
2. **Número que vira manchete.** Toda estatística é formulada como frase de
   choque com CONTRASTE embutido: "assédio aparece 7x mais que amor" · "1 em 4
   associa relacionamento a cadeia" · "a palavra mais repetida não foi medo — foi
   PAZ". Número sem contraste é planilha; número com contraste é gancho.
3. **O povo fala por ela (likes = autoridade).** Top comentários VERBATIM com
   @ + número de likes. O like transforma opinião em consenso mensurável
   ("1.202 pessoas concordaram com um POLICIAL"). Ela nunca precisa procurar
   nada: o comentário célebre JÁ ESTÁ no documento com a prova social anexada.
4. **Insight em 4 tempos, sempre:** DADO → o que está acontecendo → 2-3
   exemplos reais verbatim → **IMPLICAÇÃO em frase-lâmina quotável** ("Vocês
   não competem com outras mulheres. Competem com o SOSSEGO."). A frase-lâmina
   é o que ela leva pro vídeo — é geração de conceito-marca em série.
5. **Do dado ao vídeo em UM passo.** 10 roteiros prontos, formato 60s fixo:
   HOOK = o dado/comentário chocante (0-5s) → CONTEXTO "eu analisei" (5-15s) →
   DESENVOLVIMENTO com quotes reais (15-45s) → CTA = pergunta aberta (45-60s).
   O roteiro nasce DO estudo, na frente dela — nunca solto. (A falha de 22/07:
   roteiros sem o estudo em cima = "sem pé nem cabeça".)
6. **Ponte pro negócio.** Uma parte final traduz os dados em decisão de produto
   /posicionamento (em janeiro: o app; hoje: mentoria/BFC/publi). O estudo nunca
   é só conteúdo — valida o império.
7. **Fechamento com próximos passos + métricas.** "Grave os 10, leia comentários
   reais, posicione-se como pesquisadora, colete dados de cada vídeo novo" —
   transforma o documento em SISTEMA contínuo (cada vídeo alimenta o próximo
   estudo).
8. **O DOCUMENTO é a entrega.** ⚠️ Correção do Enrico (22/07): ela CAGOU pros
   slides — o que colou foi o MD do Claude em si (o estudo como página/documento
   denso e legível). O PDF de reportlab foi rejeitado por ser feio, mas deck
   bonito também não foi o que a ganhou. O artefato-herói é o próprio estudo
   markdown; deck/slides = acessório dispensável, só se ela pedir.

## Anti-padrões (o que matou as entregas de fev→jul)

- Roteiro solto sem o estudo-mãe visível → ela não vê o LASTRO → "cadê a estratégia?"
- Pauta "ancorada em 1 comentário" sem likes/contexto → depende dela achar o comentário → "sem pé nem cabeça"
- Relatório burocrático (cobertura, schema, pipeline) → é cozinha; ela quer a REVELAÇÃO
- Número sem contraste, insight sem frase-lâmina, quote sem like
- Voz do clone no lugar do frame de pesquisadora (o estudo NÃO é escrito na voz dela — é escrito PRA ela gravar com autoridade)

## A fórmula em uma linha
**Corpus completo → números-manchete com contraste → consenso verbatim com likes → insight com lâmina → roteiro com o dado como hook → ponte pro negócio → loop contínuo → entregue como DOCUMENTO (o MD é a arte).**
